# VisitingLink 🪪

A premium, futuristic SaaS website for a digital visiting card platform.

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run the development server
```bash
npm run dev
```

### 3. Open in browser
Visit [http://localhost:3000](http://localhost:3000)

## Build for Production
```bash
npm run build
npm start
```

## Features
- ✨ Smooth page loader animation
- 🖱️ Custom animated cursor with lag ring
- 🃏 3D card tilt effect on hero card
- 📜 Scroll reveal animations via Framer Motion
- 🌊 Parallax hero section
- 💎 Glassmorphism UI cards
- 🌈 Neon gradient accents
- 📱 Fully mobile responsive
- 🌙 Dark mode with grain noise overlay
